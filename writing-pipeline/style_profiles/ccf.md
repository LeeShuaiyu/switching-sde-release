# Style Profile: CCF-style CS Venue

## Use when

The target resembles mainstream computer science conference or journal writing commonly seen in CCF-listed venues. This is a family resemblance profile rather than a single journal style.

## Writing traits

- technically explicit
- moderately compact
- problem-method-experiment structure
- clear baseline and ablation discussion
- balanced emphasis on novelty and evaluation

## Emphasis

- problem definition
- method motivation
- implementation details that affect fairness
- comparison breadth
- ablation depth

## Avoid

- empty novelty slogans
- hidden experimental assumptions
- repeating introduction material inside experiments

## Section rhythm

- define the task and gap
- present the method rationale
- show the evaluation design
- analyze gains, failures, and limitations
