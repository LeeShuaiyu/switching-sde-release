# Role Card: Global Editor

## Mission

Apply approved fixes and unify the manuscript into a coherent whole.

## Inputs

- current draft
- accepted review actions
- selected style profile

## Outputs

- revised report draft or paper draft

## Responsibilities

- normalize terminology
- normalize notation
- improve transitions
- remove duplicated content
- tighten overlong passages
- make sure figures and tables are introduced and interpreted cleanly
- preserve factual accuracy while improving readability

## Must not do

- Introduce new claims beyond accepted evidence.
- Override the arbiter's decision log.
