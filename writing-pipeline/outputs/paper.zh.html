具有状态切换的随机动力系统中的切换时刻后验推断

我们研究这样一个问题：如何从较短的随机轨迹中定位单一潜在状态切换，同时量化切换位置上的不确定性。与只预测一个标量变点估计不同，本文方法在允许的内部切换索引集合上构造离散后验分布。该模型将序列编码器与候选约束的后验头结合起来，对每个可行切分比较左右轨迹片段的学习摘要。在参考线性基准上，相对于确定性神经回归，Transformer 后验模型将切换时刻误差从 2.1270 降到 1.6383，并把 \(\mathrm{F1}_{0.02}\) 从 0.0815 提升到 0.3015，同时消除了 collapse；若把骨干差异也纳入比较，最佳的 LSTM 后验结果达到 0.6227 MAE 和 0.6943 \(\mathrm{F1}_{0.02}\)。在轻度线性偏移下，后验模型仍然表现稳健；但在更强的线性偏移下，它会迅速退化并经常 collapse。在非线性 Duffing 基准上，LSTM 后验模型达到 0.3484 MAE 和 0.8367 \(\mathrm{F1}_{0.02}\)，且没有 collapse；不过，一个非线性压力设定仍然在点预测指标较强时触发了 collapse 警告。

具有状态切换的随机动力系统中的切换时刻后验推断

摘要

我们研究这样一个问题：如何从较短的随机轨迹中定位单一潜在状态切换，同时量化切换位置上的不确定性。与只预测一个标量变点估计不同，本文方法在允许的内部切换索引集合上构造离散后验分布。该模型将序列编码器与候选约束的后验头结合起来，对每个可行切分比较左右轨迹片段的学习摘要。在参考线性基准上，相对于确定性神经回归，Transformer 后验模型将切换时刻误差从 2.1270 降到 1.6383，并把 \(\mathrm{F1}_{0.02}\) 从 0.0815 提升到 0.3015，同时消除了 collapse；若把骨干差异也纳入比较，最佳的 LSTM 后验结果达到 0.6227 MAE 和 0.6943 \(\mathrm{F1}_{0.02}\)。在轻度线性偏移下，后验模型仍然表现稳健；但在更强的线性偏移下，它会迅速退化并经常 collapse。在非线性 Duffing 基准上，LSTM 后验模型达到 0.3484 MAE 和 0.8367 \(\mathrm{F1}_{0.02}\)，且没有 collapse；不过，一个非线性压力设定仍然在点预测指标较强时触发了 collapse 警告。

1. 引言

对于随机动力系统中的单次切换定位问题，把它当作普通的序列回归并不合适。真正感兴趣的对象是潜在转移时刻，而即使已经观察到整条轨迹，若干彼此接近的切分位置依然可能同时合理。因此，模型输出不仅应包含一个点估计，还应包含允许切换时刻上的不确定性分布。

本文研究的是由双状态切换随机系统生成的标量轨迹。每个样本在切换前遵循一个状态，在切换后遵循另一个状态。核心问题是：在受控分布偏移下，学习得到的后验模型能否比经典变点检测器和更简单的神经基线更准确、更稳定地定位切换时刻。

图 1 展示了该任务的歧义性。在重尾扰动或观测污染之后，观测路径在视觉上仍可能同时支持多个彼此接近的切分位置。这也是为什么我们更需要候选切换时刻上的后验分布，而不是单一的直接回归输出。

整体经验图景并不是单调乐观的。后验表述在线性名义基准上明显优于确定性神经基线，在轻度线性压力下仍然有效，在更强线性压力下会失败，而在非线性 Duffing 动力学上又重新变得很强。论文叙述必须完整保留这一模式，而不能只摘取其中最亮眼的部分。

2. 问题形式化与记号

设观测轨迹为

\[x_{1:L} = (x_1, \ldots, x_L),\]

其中 \(L\) 是有效时间步数，\(T_h\) 是物理观测时域。系统中恰好存在一个潜在变点 \(\tau\)，潜在状态序列在切换前满足 \(s_t = 0\)，在切换后满足 \(s_t = 1\)。对应的状态占据比例为

\[\rho = (\rho_0, \rho_1), \qquad \rho_0 = \tau/(L-1), \quad \rho_1 = 1-\rho_0.\]

切换时刻并不在全序列范围内搜索，而是限制在内部候选集合上。设最小片段长度为 \(m = 16\)，则允许集合为

\[\mathcal{T}(L) = \{\tau : \lfloor 0.2(L-1) \rfloor \le \tau \le \lceil 0.8(L-1) \rceil,\; \tau \ge m,\; (L-1)-\tau \ge m\}.\]

模型预测离散后验

\[q_\theta(\tau \mid x_{1:L}), \qquad \tau \in \mathcal{T}(L),\]

并据此导出最大后验估计、前三候选索引，以及 90% 最高后验密度区间。

基准族包含两类切换随机微分方程。在线性情形下，

\[dX_t = -\eta_j (X_t - \mu)\,dt + \varepsilon_j\, dL_t^{(\alpha_j)}, \qquad j \in \{0,1\},\]

它对应于切换 Ornstein-Uhlenbeck 过程，其中 \(\eta_j\) 是状态相关的漂移率，\(\varepsilon_j\) 是噪声尺度，\(\alpha_j\) 是稳定指数。在非线性情形下，

\[dX_t = (a_j X_t - b X_t^3 + c_j)\,dt + \varepsilon_j\, dL_t^{(\alpha_j)}, \qquad j \in \{0,1\},\]

它对应于切换 Duffing 过程。

3. 方法

3.1 序列编码器

输入轨迹首先被映射为潜在序列特征

\[h_{1:L} = f_\theta(x_{1:L}), \qquad h_t \in \mathbb{R}^{64},\]

其中编码器可以是三层 LSTM，也可以是带 0.1 dropout 的三层 Transformer。保留这两种骨干，是因为最强骨干会随基准而变化。

3.2 候选约束的后验头

对于每个允许的候选切分 \(\tau \in \mathcal{T}(L)\)，模型构造左右两个片段摘要，

\[m_L(\tau) = \frac{1}{\tau} \sum_{t<\tau} h_t, \qquad m_R(\tau) = \frac{1}{L-\tau} \sum_{t\ge \tau} h_t.\]

然后将这些摘要组合成与切分位置相关的特征向量

\[\phi(\tau) = [m_L(\tau),\; m_R(\tau),\; m_L(\tau)-m_R(\tau),\; m_L(\tau) \odot m_R(\tau),\; \tau/(L-1)],\]

并由多层感知机产生候选分数 \(s_\theta(\tau) = g_\theta(\phi(\tau))\)。后验分布是在允许候选上的 masked softmax：

\[q_\theta(\tau \mid x_{1:L}) = \frac{\exp(s_\theta(\tau))\,\mathbf{1}[\tau \in \mathcal{T}(L)]}{\sum_{u \in \mathcal{T}(L)} \exp(s_\theta(u))}.\]

这一步是方法的核心。模型不是对每个时间点独立分类，而是在判断某个具体切分是否能把轨迹分成两个在学习表示上共同合理的片段，分别对应切换前和切换后状态。

3.3 辅助量与训练变体

模型还设置了一个单独的头来预测占据比例 \(\rho\)，从而为切换位置引入辅助监督。最强的基线配置采用对真实变点的硬监督，并对 \(\rho\) 施加较小的辅助损失。

此外还评估了三个扩展方向。第一种是教师引导的排序蒸馏，教师后验由对每个候选切分两侧拟合简单 Ornstein-Uhlenbeck 代理模型得到。第二种是在蒸馏中加入置信度门控，使不确定的教师目标权重更低。第三种是向切分评分器中注入手工构造的片段统计量，如片段均值和标准差。这些扩展在分析上有价值，但都不是最终最强的配置。

4. 实验协议

每个数据划分都包含 20,000 条训练轨迹、2,000 条验证轨迹和 2,000 条测试轨迹。神经模型训练 20 个 epoch，优化器为 AdamW，学习率 \(10^{-3}\)，权重衰减 \(10^{-4}\)，批量大小为 128，并采用 1.0 的梯度裁剪。温度缩放在验证集上拟合，然后再报告经过校准的后验指标。所有神经结果都对三个随机种子取平均。

基线集合包括 CUSUM 似然比检测器、离线贝叶斯变点检测器、切换隐马尔可夫模型、plug-in 分裂后验基线、确定性神经回归，以及分割式神经后验基线。之所以保留这组基线，是因为它们能够区分“后验变点建模”的收益与“仅仅使用序列编码”的收益。

报告的指标包括切换时刻平均绝对误差 \(\mathrm{MAE}_\tau\)、容忍窗口检测分数 \(\mathrm{F1}_{0.02}\)（容忍宽度为轨迹长度的 2%）、校准后的负对数似然、校准后的期望校准误差、前三命中率 \(H_3\)，以及 90% 区间覆盖率。我们还跟踪一个 collapse 指标，

\[c = \mathbf{1}\{\mathrm{NLL}_{\mathrm{cal}} > 5.30 \;\text{or}\; r_{\mathrm{teach}} < 0.15 \;\text{or}\; H_3 < 0.025\},\]

其中 \(r_{\mathrm{teach}}\) 是模型后验与辅助教师后验之间的秩相关。这个诊断刻意设置得较严格：模型即便拥有不错的平均点预测精度，只要后验质量明显退化，仍应被视为不可靠。

该指标中的阈值来自发布版评测脚本，因此我们把它们解释为仓库定义的诊断常数，而不是普适统计阈值。由于发布基准表只提供种子平均值，没有给出置信区间，鲁棒性比较也应被读作方向性证据，而不是区间估计意义下的显著差异。

5. 结果

5.1 参考线性基准

表 2 报告了参考线性基准结果。最干净的受控比较是 Transformer 这一组：相对于确定性神经回归，后验变点模型将 MAE 从 2.1270 降到 1.6383，并把 \(\mathrm{F1}_{0.02}\) 从 0.0815 提升到 0.3015，同时 collapse 率从 1 降到 0。分割式神经后验同样发生了 collapse。如果扩大比较范围，最强的整体神经结果是 LSTM 后验模型，其 MAE 为 0.6227、\(\mathrm{F1}_{0.02}\) 为 0.6943，且没有 collapse；但这一更宽松的比较同时改变了骨干和头部，因此它更适合被理解为模型家族层面的上限空间，而不是头部表述本身的纯收益。

5.2 线性压力鲁棒性

线性压力基准暴露出一种明显不均匀的鲁棒性剖面。在线性场景 1 下，LSTM 后验模型仍然表现强劲，达到 \(\mathrm{MAE}_\tau = 0.8854\)、\(\mathrm{F1}_{0.02} = 0.5395\)，且没有 collapse，显著优于最强的简单基线。而在线性场景 2 和 线性场景 3 下，最好的后验模型分别退化到 3.1398 和 2.7924 MAE，并出现 collapse。在这些设定中，CUSUM 检测器的 MAE 低于后验模型。

图 3 强调，该方法并不具备统一鲁棒性。它能够较好迁移到“重尾加脉冲”的轻度偏移，但无法有效应对低可分性与不规则观测这两类偏移。

5.3 非线性鲁棒性

非线性 Duffing 基准对该方法更为友好。图 4 已经显示，非线性场景中的误差趋势比线性压力套件更平缓。在参考非线性基准上，LSTM 后验模型达到 0.3484 MAE 和 0.8367 \(\mathrm{F1}_{0.02}\)，且没有 collapse；Transformer 后验模型也保持了较强表现，达到 0.4991 MAE 和 0.7573 \(\mathrm{F1}_{0.02}\)。在非线性场景 2 和 非线性场景 3 下，后验模型家族保持零 collapse，并在点预测精度上明显领先于经典基线。

主要保留意见来自 非线性场景 1。在该设定下，两种骨干的点预测指标都很强，但 collapse 指标仍被触发。因此，图 4 必须和表 4 一起阅读：视觉上较小的误差变化，并不保证后验本身仍然健康。相比其他场景，这一设定最清楚地将点预测精度与后验可靠性区分开来。

5.4 消融研究

消融研究支持一种克制的解释。最强的 Transformer 配置其实是最简单的那一个：只使用硬后验监督。它的参考线性指标为 1.6383 MAE 和 0.3015 \(\mathrm{F1}_{0.02}\)。教师引导的排序蒸馏略弱一些，为 1.6991 和 0.2895。带置信度门控的蒸馏进一步下降到 1.7269 和 0.2872。手工片段统计增强则明显不稳定：误差上升到 2.6786，并在三次运行中的两次发生 collapse。因此，主要的经验贡献在于候选约束的后验表述本身，而不是更复杂的辅助变体。

6. 讨论

现有证据支持三个判断。第一，与其把任务视为普通回归，不如把它写成变点的后验推断。第二，候选约束的后验建模显著改善了线性名义场景中的性能与神经稳定性。第三，同一表述在非线性切换动力学上依然有效。

这些证据也清楚地限定了结论边界。该方法并不能解决严重的线性偏移，尤其当状态可分性变差或观测过程变得不规则时更是如此。此外，如非线性场景 1 所示，点预测指标较强并不意味着后验行为健康。这些局限并不是次要细节，而是后续改进最有意义的目标。
