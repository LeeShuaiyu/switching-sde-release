# 技术报告：随机动力系统中单一切换时刻的后验推断

## 1. 执行摘要

本报告研究随机轨迹中的单次切换变点问题。每条观测序列在切换之前由一个潜在状态生成，在切换之后由另一个潜在状态生成。我们的目标不仅是推断切换索引，还要刻画该索引上的不确定性，而不是只回归一个标量预测值。

核心方法是后验变点模型。该模型使用 LSTM 或 Transformer 作为轨迹编码骨干，对每一个允许的切分点进行评分；评分方式是比较左右两个片段的学习摘要，然后将这些分数归一化为候选切换时刻上的离散后验分布。这个结构之所以重要，是因为它把点预测精度与后验质量、校准质量区分开来。

主要结论如下。在参考线性基准上，后验变点建模比确定性回归和分割式神经基线稳定得多。在包含更重尾分布和稀疏脉冲污染的轻度线性偏移下，该方法仍然表现稳健。在由低可分性或不规则观测引起的更强线性偏移下，性能会显著恶化，并出现 collapse 现象。在非线性 Duffing 基准上，该方法在点预测精度方面再次表现突出，而且通常保持稳定；不过，在一个非线性压力设定下，collapse 诊断仍然会被触发。

## 2. 问题表述与记号

设观测轨迹为

```math
x_{1:L} = (x_1, \ldots, x_L),
```

其中 `L` 表示有效轨迹长度，`T_h` 表示物理观测时域。系统中恰好存在一个潜在变点 `\tau`，潜在状态序列在切换前满足 `s_t = 0`，在切换后满足 `s_t = 1`。相应的状态占据比例为

```math
\rho = (\rho_0, \rho_1), \qquad \rho_0 = \tau/(L-1), \quad \rho_1 = 1 - \rho_0.
```

切换时刻被限制在内部候选集合上。令最小片段长度为 `m = 16`，则允许集合为

```math
\mathcal{T}(L) = \{\tau : \lfloor 0.2(L-1) \rfloor \le \tau \le \lceil 0.8(L-1) \rceil,\; \tau \ge m,\; (L-1)-\tau \ge m\}.
```

估计器输出后验分布

```math
q_\theta(\tau \mid x_{1:L}), \qquad \tau \in \mathcal{T}(L),
```

据此可导出最大后验估计、前三候选索引，以及 90% 最高后验密度区间。

本文考虑两类切换随机微分方程。在线性族中，

```math
dX_t = -\eta_j (X_t - \mu)\,dt + \varepsilon_j\, dL_t^{(\alpha_j)}, \qquad j \in \{0,1\},
```

这对应于切换 Ornstein-Uhlenbeck 过程。在非线性族中，

```math
dX_t = (a_j X_t - b X_t^3 + c_j)\,dt + \varepsilon_j\, dL_t^{(\alpha_j)}, \qquad j \in \{0,1\},
```

这对应于切换 Duffing 过程。

## 4. 方法

### 4.1 架构

模型首先将输入轨迹映射为潜在序列特征

```math
h_{1:L} = f_\theta(x_{1:L}), \qquad h_t \in \mathbb{R}^{64},
```

其中编码器可以是三层 LSTM，也可以是带 0.1 dropout 的三层 Transformer。

对于每个候选点 `\tau \in \mathcal{T}(L)`，后验头都会构造左右两个片段的摘要，

```math
m_L(\tau) = \frac{1}{\tau}\sum_{t<\tau} h_t,
\qquad
m_R(\tau) = \frac{1}{L-\tau}\sum_{t\ge \tau} h_t,
```

并将它们组合成与切分位置相关的特征向量

```math
\phi(\tau) = [m_L(\tau),\; m_R(\tau),\; m_L(\tau)-m_R(\tau),\; m_L(\tau) \odot m_R(\tau),\; \tau/(L-1)].
```

多层感知机产生候选分数 `s_\theta(\tau) = g_\theta(\phi(\tau))`，而后验分布则是对允许候选集合上的 masked softmax：

```math
q_\theta(\tau \mid x_{1:L}) = \frac{\exp(s_\theta(\tau))\,\mathbf{1}[\tau \in \mathcal{T}(L)]}{\sum_{u \in \mathcal{T}(L)} \exp(s_\theta(u))}.
```

这种“候选约束”的构造是本文最核心的建模贡献。模型关注的不是逐时刻独立分类，而是某个具体切分是否能把轨迹划分为在表示空间中共同合理的切换前片段与切换后片段。

### 4.2 辅助头与训练变体

模型另设一个头来预测占据比例 `\rho`，为切换位置提供附加监督信号。最强的基线配置采用对真实变点的硬监督，同时对 `\rho` 加入一个较小的辅助损失。

基准套件还评估了三个扩展版本：一是利用简单 Ornstein-Uhlenbeck 代理模型构造 plug-in 后验，并进行教师引导的排序蒸馏；二是在蒸馏中加入置信度门控；三是加入手工设计的片段统计量增强。这些变体对诊断有帮助，但并不是最终表现最好的配置。

## 5. 实验协议

每个数据划分都包含 20,000 条训练轨迹、2,000 条验证轨迹和 2,000 条测试轨迹。神经模型使用 AdamW 训练 20 个 epoch，学习率 `10^{-3}`，权重衰减 `10^{-4}`，批量大小为 128，梯度裁剪阈值为 1.0。温度缩放在验证集上拟合，然后再报告经过校准的后验指标。所有神经结果都对三个随机种子取平均。

基线集合包括 CUSUM 似然比检测器、离线贝叶斯变点检测器、切换隐马尔可夫模型、plug-in 分裂后验基线、确定性神经回归，以及分割式神经后验。这种覆盖范围很重要，因为它可以把“后验变点建模”的价值与“仅仅使用序列编码器”的价值区分开来。

主要指标包括切换时刻平均绝对误差 `\mathrm{MAE}_\tau`、容忍窗口检测分数 `\mathrm{F1}_{0.02}`、校准后的负对数似然、校准后的期望校准误差、前三命中率 `H_3`，以及 90% 区间覆盖率。报告还跟踪一个 collapse 指标，

```math
c = \mathbf{1}\{\mathrm{NLL}_{\mathrm{cal}} > 5.30 \;\text{or}\; r_{\mathrm{teach}} < 0.15 \;\text{or}\; H_3 < 0.025\},
```

其中 `r_{\mathrm{teach}}` 是模型后验与辅助教师后验之间的秩相关系数。这个诊断刻意设置得较严格：即便模型的平均点误差还能接受，只要后验本身明显退化，仍会被判定为不可靠。

这些阈值继承自发布版基准脚本，因此本报告将其视为仓库定义的诊断常数，而不是普适性的统计阈值。由于发布表格只给出了种子平均值，没有提供置信区间或误差条，鲁棒性比较也应被理解为方向性的证据，而不是带区间估计的差异结论。

## 7. 线性压力结果

线性压力基准揭示出一种明显不均匀的鲁棒性剖面。在线性场景 1 下，LSTM 后验模型仍然表现强劲，达到 `\mathrm{MAE}_\tau = 0.8854`、`\mathrm{F1}_{0.02} = 0.5395`，且没有出现 collapse，显著优于最强的简单基线。而在线性场景 2 和 线性场景 3 下，最好的后验模型分别退化到 3.1398 和 2.7924 MAE，并出现 collapse。在这些设定中，CUSUM 检测器的 MAE 反而低于后验模型。

## 9. 消融与解释

消融结果表明，最强的 Transformer 变体反而是最简单的那一个：不使用更复杂教师引导扩展、只采用硬后验监督。它在线性参考基准上的指标为 1.6383 MAE 和 0.3015 `\mathrm{F1}_{0.02}`。教师引导的排序蒸馏略微削弱这些数值，降为 1.6991 和 0.2895。带置信度门控的蒸馏进一步降为 1.7269 和 0.2872。片段统计增强则表现不稳定，误差升至 2.6786，并在三次运行中的两次发生 collapse。

因此，主要贡献是候选约束的后验表述本身。教师引导蒸馏和手工片段统计更适合被视为探索性扩展，而不是已经验证过的核心方法。

## 10. 总体结论

一旦将任务写成变点的后验推断，完整的经验图景就变得内部一致。该方法在线性参考基准上明显优于更简单的神经基线，对轻度线性压力具有真实鲁棒性，对更强线性偏移则明显脆弱，而在非线性切换动力学上又重新展现出很强的表现。严格的 collapse 诊断避免了报告对最强非线性结果作出过度乐观的表述。

因此，对这项工作的自包含总结应强调两点。第一点是建模思想：比较候选切分左右两侧的片段，并在允许的切换时刻上建立后验分布。第二点是局限性：鲁棒性高度依赖于偏移机制，尤其依赖于状态可分性与观测规则性。这两点也是后续改进最值得投入的方向。
